The Path to Enlightment
These instructions are for *patforms. We also have Windows instructions.

You can run the tests by caling the path_to_enlighnenment.rb file.

In your terminal, while in the ruby.koans directory, type:

[ ruby_koans ] $ ruby path_to_enlightment.rb

Red, Green, Refactor

In test-driven development (TDD) the mantra has always been red: write a failing test and run it, green: make the test pass, and refactor: look at the code and see if you can make it any better.
